# AutiCare AI

## Project Info

AutiCare AI is a comprehensive platform designed to connect parents, doctors, and therapists for seamless autism care management.

## Getting Started

To run this project locally:

1.  **Clone the repository**
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Install Dependencies**
    ```sh
    cd auticareai/frontend
    npm install
    ```

3.  **Start Development Server**
    ```sh
    npm run dev
    ```

## Features

*   **Role-Based Access**: Specialized dashboards for Parents, Doctors, and Therapists.
*   **Child Profile Management**: Track progress, medical history, and therapy sessions.
*   **Report Generation**: Generate detailed PDF reports for assessments.
*   **Secure Authentication**: Powered by Supabase and Google OAuth.

## Tech Stack

*   **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
*   **Backend**: Node.js, Express
*   **Database**: Supabase (PostgreSQL)
*   **Auth**: Supabase Auth, Google OAuth
